AnyText

AnyText class

A union type for any text widget that can be found in the widget tree. Specifically this includes:

  • Text
  • SelectableText
  • RichText
  • EditableText

Consolidated attributes may, depending on the text widget type, include:

  • maxLines,
  • minLines,
  • textStyle,
  • text,
  • overflow,
  • softWrap,
  • textDirection,
  • textAlign,
  • locale,
  • selectionColor
