SelectorQueries

SelectorQueries<W extends Widget> extension

Extension on Selectors<W> for advanced widget querying.

Adds methods to refine widget selections based on elements, widgets, and their properties. Filters are applied during the snapshotting of the selector, enabling precise widget selection.

Methods:

  • whereElement: Filter on Element properties.
  • whereWidget: Filter based on widget properties.
  • whereWidgetProp: Filter widgets by a specific property.
  • whereElementProp: Filter elements by a specific property.
  • whereRenderObjectProp: Filter by render object properties.

Each method requires a predicate and a description for clear error messages and precise selection.
