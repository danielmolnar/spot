fontSize property

Subject<double> fontSize

A Subject<double> representing the fontSize of the TextStyle. If the fontSize is not explicitly set, it defaults to 14.

Implementation

Subject<double> get fontSize {
  return context.nest(
    () => ['has fontSize'],
    (it) => Extracted.value(it.fontSize ?? 14),
  );
}
